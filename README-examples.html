Create access token

This sample creates an access token with your client credentials.

Don't forget you need to configurate your client credentials in config.php file (php/src/config.php) before trying our examples!!

The files for this sample are:

File

Description

createAccessToken.html

This file contains a button to run example.

Calls createAccessTokenSample.php

createAccessTokenSample.php

This example is used to connect with Auth service sending client credentials. It shows accessData structure.

Called by createAccessToken.html

Calls authService.php

Uses config.php
