Services description

Class

Service

Description

AuthService

create_access_data

This Service receives the client_credentials and starts the request to oAuth API. On success returns an accessData Object, otherwise returns an exception.

Example of use:

$authService = new AuthService();

$accessData= $authService->create_access_data(CLIENT_ID,CLIENT_SECRET);

refresh_access_token

This Service receives the client_credentials and refresh_token to start the request to oAuth API. On success returns an accessData Object with autorization data(included the access_token refreshed),otherwise returns an exception.

Example of use:

$authService = new AuthService();

$accessData= $authService->refresh_access_token(CLIENT_ID,CLIENT_SECRET, $refreshToken);

CheckoutService

create_checkout_preference

This Service receives an access token and checkoutPreferenceData Object to execute a Post request to Preference API. On success returns a checkoutPreference Object, otherwise returns an exception. Once you created the checkout preference, you can use init_point url to open checkout flow. More information here

Example of use:

$checkoutService = new CheckoutService();

$checkoutPreference=$checkoutService->create_checkout_preference($checkoutPreferenceData,$access_token);

get_checkout_preference

This Service receives an access token and a checkoutPreference Id and starts a Get request to Preference API. On success returns a checkoutPreference Object, otherwise returns an exception.

Example of use:

$checkoutService = new CheckoutService();

$checkoutPreference=$checkoutService->get_checkout_preference($preference_id,$access_token);

update_checkout_preference

This Service receives a checkoutPreferenceData Object and access token to start a Put request to Preference API. On success returns a 201 status,otherwise returns an exception.

Example of use:

$checkoutService = new CheckoutService();

$checkoutPreferenceStatus=$checkoutService->update_checkout_preference($preference_id,$checkoutPreferenceData,$access_token);

CollectionService

search_collection

This Service receives an array of filters and an access token to execute a Search request to Collection API. On success return a list of collection Objects, otherwise returns an exception.

Example of use:

$collectionService = new CollectionService();

$collectionList=$collectionService->search_collection($filters,$access_token);

Installing Samples

Copy the samples folder, to the docroot of the WebFrefresh Server.

Before you can use our samples it's totaly necesary you configurate the config.php file(php/src/config.php). This file contains your credentials needed by oAuth API.
